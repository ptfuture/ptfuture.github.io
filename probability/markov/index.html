Discrete Time Markov Chain

Consider a stochastic process {Xn,n=0,1,2,...}\{ X_n, n = 0, 1, 2,...\}{Xn​,n=0,1,2,...} that takes finite or countable number of possible values.

If Xn=iX_n=iXn​=i we say the process is in state i at time n. We suppose that when the process is in state i, the probability that the process will enter state jjj is Pij\boldsymbol{P}_{ij}Pij​.

Define the discrete time markov chain as:

P(Xn+1=j∣Xn=i,Xn−1=in−1,...,X1=i1,X0=i0)=P(Xn+1=j∣Xn=i)=PijP(X_{n+1} = j | X_n = i, X_{n-1} = i_{n-1}, ..., X_1 = i_1, X_0 = i_0) = P(X_{n+1} = j | X_n = i) = \boldsymbol{P}_{ij} P(Xn+1​=j∣Xn​=i,Xn−1​=in−1​,...,X1​=i1​,X0​=i0​)=P(Xn+1​=j∣Xn​=i)=Pij​

Markov chain is a Memoryless Process.

We say jjj is transient if ρjj<1\rho_{jj} \lt 1ρjj​<1. That is there's chance 1−ρjj1-\rho_{jj}1−ρjj​ that the process will never return to state jjj.

If a process is 100% sure to return back to its origin state, then the average of times of returns is infinite. If a process is not 100% sure to return to its origin, then the average of times of returns is finite.
