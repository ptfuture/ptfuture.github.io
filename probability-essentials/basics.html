Unqiueness of Moment Generating Function¶

Let the CDF of \(\boldsymbol{X}, \boldsymbol{Y}\) are \(F_{\boldsymbol{X}}(x)\) and \(F_{\boldsymbol{Y}}(y)\), MGF are \(M_{\boldsymbol{X}}(t)=E[e^{t{\boldsymbol{X}}}]\) and \(M_{\boldsymbol{Y}}(t)=E[e^{t{\boldsymbol{Y}}}]\). Then \(F_{\boldsymbol{X}}(z)=F_{\boldsymbol{Y}(z)}\) for all \(z\in R\) iff \(M_{\boldsymbol{X}}(t)=M_{\boldsymbol{Y}}(t)\) for all \(t\in (-h,h)\) for some \(h \gt 0\).

Independence By MGF¶

\(X = [X_1, X_2]'\) are independent iff \(M(\boldsymbol{t}) = E[e^{\boldsymbol{t}'\boldsymbol{X}}] = M_{X_1}(t_1)M_{X_2}(t_2)\)

Proof:

\(X = [X_1, X_2]'\) independent, then \(M(\boldsymbol{t}) \implies E[e^{\boldsymbol{t}'\boldsymbol{X}}] = M_{X_1}(t_1)M_{X_2}(t_2)\).

\(M(\boldsymbol{t}) \impliedby E[e^{\boldsymbol{t}'\boldsymbol{X}}] = M_{X_1}(t_1)M_{X_2}(t_2)\), by uniqueness of MGF, \(f(x,y)\) of \(M(\boldsymbol{t})\) equals \(f_xf_y\) of \(M_{X_1}(t_1)M_{X_2}(t_2)\)

Pearson correlation coefficient between -1 and 1¶

\(\rho = \frac{Cov(X,Y)}{\sigma_{\boldsymbol{X}}\sigma_{\boldsymbol{Y}}} \in [-1, 1]\)

Proof:

By Holder’s inequality (Generalization of Cauch Schwarz inequality) in \(L^2\) space: \(|Cov(X,Y)| \le \sigma_{\boldsymbol{X}}\sigma_{\boldsymbol{Y}}\)

Commonly used Properties and Formulas¶

  1. \(Cov(cX, Y) = cCov(X,Y)\)

  2. \(Cov(X+Y, Z) = Cov(X, Z) + Cov(Y, Z) = Cov(X, Y+Z)\)

  3. \(\displaystyle Cov(\sum_{i=1}^n{X_i}, \sum_{j=1}^m{Y_j}) = \sum_{i=1}^n\sum_{j=1}^mCov(X_i, Y_j)\)

  4. \(\displaystyle Var(\sum_{i=1}^nX_i) = Cov(\sum_{i=1}^nX_i, \sum_{j=1}^nX_j) = \sum_{i=1}^{n}Var(X_i) + 2\sum_{i=1}^n\sum_{j\gt{i}} Cov(X_i, Y_j)\) If \(X_i\) are has common \(\mu, \sigma^2\), then \(\displaystyle Var(\sum_{i=1}^nX_i) = n\sigma^2 + n(n-1)\rho \sigma^2\)

  5. If \(X_1, X_2, ..., X_n\) are iid, then, \(Cov(\bar{X}, X_i-\bar{X}) = 0\). Proof: $\( \begin{aligned} \displaystyle Cov(\bar{X}, X_i-\bar{X}) &= Cov(\bar{X}, X_i) - Cov(\bar{X}, \bar{X}) \\ &= \frac{1}{n}Cov(X_i + \sum_{j\ne{i}}X_j, X_i) - Var(\bar{X}) &= \frac{\sigma^2}{n} - \frac{\sigma^2}{n} = 0 \end{aligned} \)$

  6. If \(Z ~ N(0,1)\), then \(Z^2 \sim \chi^2(1)\). Calculate \(P(Z^2 \le z)\). Same, if \(Z_1, Z_2,..., Z_n\) are iid standard normal, then \(\displaystyle\sum_{i=1}^nZ_i^2 \sim \chi^2(n)\)

  7. \(E_Y[E_X[X|Y]] = \int\int{xf(x|y)} g_Y(y) dxdy = \int\int{x\frac{f(x,y)}{f_Y(y)}} f_Y(y) dxdy = \int\int{xf(x,y)} dxdy = E_X[X]\)

  8. \(\displaystyle Var[X] = E_Y[Var[X|Y]] + Var[E_X[X|Y]]\), \(Var[E_X[X|Y]] \le Var[X]\) Proof: $\( \begin{aligned} E[Var[X|Y]] &= E[E[X^2|Y]] - E[E[X|Y]^2] = E[X^2] - E[E[X|Y]^2] \\ Var[E[X|Y]] &= E[E[X|Y]^2] - E[X]^2 \end{aligned} \)$

Uniform \(u(a,b)\)¶

  1. \(f(x)=\frac{1}{b-a}\)

  2. \(\phi(t) = \frac{e^{tb}-e^{ta}}{t(b-a)}\)

  3. \(\mu=\frac{a+b}{2}\)

  4. \(\sigma^2=\frac{(b-a)^2}{12}\)

Binomial \(b(n,p)\)¶

  1. \(p(x)=\binom{n}{x}p^x(1-p)^{n-x}\) where \(x=0,1,2,\dots,n\)

  2. \(\phi(t)=(pe^t+(1-p))^n\)

  3. \(\,u = np\)

  4. \(\sigma^2=np(1-p)\)

Geometric \(g(p)\)¶

  1. \(p(k)=p(1-p)^{k-1}\) where \(k=1,2,\dots\)

  2. \(\phi(t) =\frac{pe^t}{1-(1-p)e^t}\)

  3. \(\mu=\frac{1}{p}\)

  4. \(\sigma^2=\frac{1-p}{p^2}\)

Poisson \(\rho(\lambda)\)¶

  1. \(p(x)=e^{-\lambda}\frac{\lambda^x}{x!}\) where \(x=0,1,2,\dots\)

  2. \(\phi(t)=e^{\lambda(e^t-1)}\)

  3. \(\mu=\lambda\)

  4. \(\sigma^2=\lambda\)

Poisson usually used to model probability of \(x\) changes in each interval of of length \(h\) with the following postulates :

  1. \(g(1, h)=\rho h + o(h)\), \(h \gt 0\), \(\rho > 0\) constant. \(\rho\) is called poisson constant.

  2. \(\sum_{x=2}^\infty g(x,h)=o(h)\)

  3. The number of changes in non-overlapping intervals are independent.

  4. \(g(x, 0) = 0\)

It can be proven that \(g(0, h)=e^{-\rho h}\), \(g(x,h)=\frac{(\rho h)^xe^{-\rho h}}{x!}\) for \(x \gt 0\)

Theorem¶

Let \(X_1, X_2,\dots,X_n\) iid \(\sim \rho(\lambda_i)\), then \(Y=\sum_{i=1}^nX_i \sim \rho(\sum_{i=1}^n\lambda_i)\)

Proof: Use MGF.

Gamma \(\Gamma(\alpha, \beta)\)¶

  1. \(f(x) = \frac{x^{\alpha-1}e^{-\frac{x}{\beta}}}{\Gamma(\alpha-1)\beta^{\alpha}}\), where \(0 \lt x \lt \infty\)

  2. \(\phi(t) = \frac{1}{(1-\beta t)^\alpha}\), where \(t \lt \frac{1}{\beta}\)

  3. \(\mu = \alpha\beta\)

  4. \(\sigma^2 = \alpha\beta^2\)

Let \(W\) be the time that is needed to observe exactly \(\alpha\) events, assuming the poisson postulates also holds for those events(\(\lim\limits_{h \to 0}\frac{g(1, h)}{h} = \rho\)).

\(P(W \gt w)\) means there are less than \(\alpha\) events occurred in the time interval of \(w\). That is \(P(W \gt w) = \sum_{k=1}^{\alpha-1}\frac{(\rho w)^ke^{-\rho w}}{k!}\)

Then we derive the Gamma distribution as follows:

\[\begin{split} \begin{align} P(W \le w) & = 1 - P(W \gt w) = 1 - \sum_{k=1}^{\alpha-1}\frac{(\rho w)^ke^{-\rho w}}{k!} \\ & = 1 - \int_{\rho w}^\infty \frac{x^{\alpha-1}e^{-x}}{\Gamma(\alpha-1)}dx \\ & = \int_{-\infty}^{\rho w}\frac{x^{\alpha-1}e^{-x}}{\Gamma(\alpha-1)}dx \\ & = \int_{-\infty}^w \frac{(\rho y)^{\alpha-1}e^{-\rho y}}{\Gamma(\alpha-1)}\rho dy \\ & = \int_{-\infty}^w \frac{\rho^{\alpha}y^{\alpha-1}e^{-\rho y}}{\Gamma(\alpha-1)} dy \\ & = \int_{-\infty}^w \frac{y^{\alpha-1}e^{-\frac{y}{\beta}}}{\Gamma(\alpha-1)\beta^{\alpha}} dy \end{align} \end{split}\]

Note: let \(\beta = \frac{1}{\rho}\)

Exponential \(e(\lambda)\)¶

When \(\alpha = 1\), we derives exponential distribution with \(\lambda = \rho\), where \(\rho\) is the poisson constant.

  1. \(f(x)=\lambda e^{- \lambda x}\) where \(x > 0\)

  2. \(\phi(t) = \frac{\lambda}{\lambda-t}\)

  3. \(\mu=\frac{1}{\lambda}\)

  4. \(\sigma^2=\frac{1}{\lambda^2}\)

Chi-square \(\chi^2(r)\)¶

When \(\beta=2\), \(\alpha=\frac{r}{2}\), we got \(\chi^2(r)\) distribution.

  1. \(f(x)=\frac{x^{\frac{r}{2}-1}e^{-\frac{x}{2}}}{\Gamma(\frac{r}{2}-1)2^{\frac{2}{2}}}\)

  2. \(\phi(t) = \frac{1}{(1-2 t)^\frac{r}{2}}\), where \(t \lt \frac{1}{2}\)

  3. \(\mu = r\)

  4. \(\sigma^2 = 2r\)

Beta \(\beta(\theta, \zeta)\)¶

  1. \(f(x)=\frac{x^{\theta + \zeta -1}e^{-x}}{\Gamma(\theta + \zeta)}\)

  2. \(\phi(t)\)

  3. \(\mu=\frac{\theta}{\theta+\zeta}\)

  4. \(\sigma^2=\frac{\theta\zeta}{(\theta+\zeta+1)(\theta+\zeta)^2}\)

Student Theorem¶

Let \(X_1, X_2, \dots, X_n\) iid \(N(\mu, \sigma^2)\). Then:

  1. \(\bar{X} \sim N(\mu, \frac{\sigma^2}{n})\)

  2. \(\bar{X}\) and \(S^2\) are independent

  3. \(\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)\)

  4. \(T=\frac{\frac{(\bar{X}-\mu)}{\frac{\sigma}{\sqrt{n}}}}{\sqrt{\frac{(n-1)S^2}{\sigma^2(n-1)}}} =\frac{\sqrt{n}(\bar{X}-\mu)}{S} \sim t(n-1)\)

Proof:

  1. Prove by MGF, or by the fact that linear combination of iid normals are also normal with \(E[\sum{\frac{X_i}{n}}] = n\frac{\mu}{n}\) and \(Var(\sum{\frac{X_i}{n}})=n\frac{\sigma^2}{n^2}\)

  2. Use the fact that \(\bar{X}\) and \({X_1-\bar{X}, X_2-\bar{X},...,X_n-\bar{X}}\) are all normal and the fact that the covoriance of \(\bar{X}\) and the random vector \({X_i-\bar{X}}\) is zero, they are indenpdent, so is the functino of independent variables.

  3. Refactor sample variance using \(\sum{(\frac{(X_i-\mu) + (\mu-\bar{X})}{\sigma})^2}\) and use independence of \(S^2, \bar{X}\) and mgf to derive that the final mgf is the mgf of \(\chi^2(n-1)\)

  4. Is a result of 1-3 and the definition of student distribution.
